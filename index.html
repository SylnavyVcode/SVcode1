les commandes git

git init pour initialiser mon git. git --version pour voir la version de git

Git Branch avec le nom de la branche pour créer une nouvelle branche différente de la branche main ou master

git branch pour voir tout les branches du projet et git checkout avec le nom de la branche auquel nous voulons faire nos modification
